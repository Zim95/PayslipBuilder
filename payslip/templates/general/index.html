Employee Details Format

What fields will each employee have

Compulsary field: name (Already exists)

Request Type: POST

Request Data: { fieldname: [fieldtype] }

Allowed Field Types:

  • Integer eg: Request Data:{ "fieldnamexyz": "Integer" }
  • Character eg: Request Data: { "fieldnamexyz": "Character", "size": 255 }
  • Dropdown eg: Request Data: { "fieldnamexyz": "Dropdown", "choices": ["a","b","c"] }
  • Date eg: Request Data: { "fieldnamexyz": "Date", }
  • Operation eg: Request Data: { "fieldnamexyz": "Operation", "operation": "fieldnamexyz + fieldnamexyz" }
